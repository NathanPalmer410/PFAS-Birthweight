Documentation

Overview

The pfas_birthweight package provides a suite of functions for loading, cleaning, merging, and analyzing PFAS drinking water contamination data alongside CDC birth weight outcomes. This page documents all available functions.

Pipeline Functions

build_pfas_birthweight_panel()

The primary function that runs the entire pipeline end-to-end and returns a county-year panel ready for analysis.

Signature:

build_pfas_birthweight_panel()

Returns:

  • pd.DataFrame: A panel DataFrame with the following columns:
Column Description
FIPS 5-digit county FIPS code
STATE 2-letter state abbreviation
COUNTY_SERVED County name
year Year (2016–2024)
births Number of births
avg_birth_weight Average birth weight (grams)
PFAS_county Population-weighted mean PFAS concentration (ng/L)

Example:

from pfas_birthweight import build_pfas_birthweight_panel

panel = build_pfas_birthweight_panel()
print(panel.head())

load_pfas()

Loads and cleans the raw PFAS sample data from the bundled Excel file, replacing non-detects with MRL/2.

Signature:

load_pfas()

Returns:

  • pd.DataFrame: Cleaned sample-level PFAS data with a result column in ng/L

collapse_pfas_to_pws()

Aggregates sample-level PFAS measurements to one row per public water system.

Signature:

collapse_pfas_to_pws(pfas_df)

Parameters:

  • pfas_df (pd.DataFrame): Output of load_pfas()

Returns:

  • pd.DataFrame: One row per PWS ID with mean PFAS concentration and population served

load_geo()

Loads the EPA SDWA Geographic Areas file, keeping only county-level service area rows (AREA_TYPE_CODE == "CN").

Signature:

load_geo()

Returns:

  • pd.DataFrame: Filtered geographic areas with PWSID and COUNTY_SERVED columns

build_county_pfas()

Merges PWS-level PFAS data with the county mapping and computes population-weighted average PFAS per county.

Signature:

build_county_pfas(pws_pfas, geo_df)

Parameters:

  • pws_pfas (pd.DataFrame): Output of collapse_pfas_to_pws()
  • geo_df (pd.DataFrame): Output of load_geo()

Returns:

  • pd.DataFrame: One row per (STATE, COUNTY_SERVED) with a PFAS_county column

load_crosswalk()

Loads the Census national county FIPS crosswalk and standardizes county names by removing suffixes like “County”, “City”, and “Parish”.

Signature:

load_crosswalk()

Returns:

  • pd.DataFrame: Crosswalk with STATE, FIPS, COUNTY_NAME, and COUNTY_CLEAN columns

attach_fips_to_pfas()

Joins 5-digit FIPS codes onto the county-level PFAS data using the Census crosswalk.

Signature:

attach_fips_to_pfas(county_pfas, crosswalk)

Parameters:

  • county_pfas (pd.DataFrame): Output of build_county_pfas()
  • crosswalk (pd.DataFrame): Output of load_crosswalk()

Returns:

  • pd.DataFrame: county_pfas with a FIPS column attached

load_birth_weights()

Loads the CDC WONDER birth weight data, dropping suppressed cells and CDC footer rows.

Signature:

load_birth_weights()

Returns:

  • pd.DataFrame: Panel data with columns FIPS, year, births, and avg_birth_weight

build_panel()

Merges the birth weight panel with county-level PFAS exposure data on FIPS code.

Signature:

build_panel(bw_df, pfas_fips)

Parameters:

  • bw_df (pd.DataFrame): Output of load_birth_weights()
  • pfas_fips (pd.DataFrame): Output of attach_fips_to_pfas()

Returns:

  • pd.DataFrame: Final panel sorted by FIPS and year

Analysis Functions

plot_eda()

Produces two side-by-side exploratory plots of the panel data.

Signature:

plot_eda(panel)

Parameters:

  • panel (pd.DataFrame): Output of build_pfas_birthweight_panel()

Plots produced:

  • Scatter plot of PFAS concentration vs average birth weight with an OLS trend line
  • Average birth weight over time (2016–2024) by state

Example:

from pfas_birthweight import build_pfas_birthweight_panel, plot_eda

panel = build_pfas_birthweight_panel()
plot_eda(panel)

run_regression()

Runs an OLS regression of average birth weight on PFAS concentration with year fixed effects and standard errors clustered by county.

Signature:

run_regression(panel)

Parameters:

  • panel (pd.DataFrame): Output of build_pfas_birthweight_panel()

Returns:

  • RegressionResultsWrapper: A fitted statsmodels results object

Model:

\[\text{BirthWeight}_{ct} = \beta \cdot \text{PFAS}_{c} + \gamma_t + \varepsilon_{ct}\]

Where \(\gamma_t\) represents year fixed effects. County fixed effects are excluded by design since PFAS exposure is time-invariant and would be fully absorbed.

Example:

from pfas_birthweight import build_pfas_birthweight_panel, run_regression

panel = build_pfas_birthweight_panel()
model = run_regression(panel)
